# 🚀 Grokking Algorithms - Java Implementations

This repository documents my journey of learning fundamental algorithms and data structures. I am reading the book *"Grokking Algorithms"* by Aditya Bhargava and translating the original Python examples into **Java** from scratch. 

As a first-year Computer Engineering student at Hacettepe University, my goal with this repository is to build a strong foundation in algorithmic thinking, understand Big O notation, and practice clean coding principles.

## 🛠️ Tech Stack
* **Language:** Java 
* **Concepts Covered:** Binary Search, Sorting Algorithms, Recursion, Hash Tables, Graph Algorithms, Dynamic Programming.

## 📂 Table of Contents
* **Chapter 1:** Introduction to Algorithms (Binary Search, Big O)
* **Chapter 2:** Selection Sort (Arrays vs. Linked Lists)
* **Chapter 3:** Recursion (Call Stack)
* *(I will update this list as I progress through the book!)*

## 💡 What I'm Learning
For every algorithm I implement, I make sure to include:
1. The **Time Complexity** (Big O notation) as a comment in the code.
2. The **Space Complexity**.
3. A brief explanation of how the algorithm works in my own words.
